## v1.1 Requirements — Final Status

### KV — データ層（Upstash Redis移行）

- [x] **KV-01**: Member データを Upstash Redis で管理できる（`{name, substackId, teamId, addedAt}`）— Phase 4で実装
- [x] **KV-02**: アプリ起動時、members.json の既存データを KV にシードできる移行スクリプトがある — Phase 4で実装（scripts/seed-kv.ts）

### ADM — 管理画面

- [x] **ADM-01**: `/admin` でメンバーを追加できる（name + substackId + teamId）— Phase 6で実装
- [x] **ADM-02**: `/admin` でメンバーを削除できる — Phase 6で実装
- [x] **ADM-03**: `/admin` は Basic 認証（ENV パスワード）で保護されている — Phase 6で実装

### HEAT — WeeklyHeatmap

- [x] **HEAT-01**: トップページに直近7日間 × 全メンバーのヒートマップを表示できる — Phase 5で実装
- [x] **HEAT-02**: ヒートマップ左端列にメンバー名を表示し、クリックで `/member/{substackId}` へ遷移できる — Phase 5で実装
- [x] **HEAT-03**: ヒートマップは 7日間投稿量降順・同数の場合は addedAt 昇順でソートされる — Phase 5で実装
- [x] **HEAT-04**: team-id でヒートマップ行をフィルタリングできる（URL param `?team=teamId`）— Phase 6で実装

### TIP — リッチ Tooltip

- [x] **TIP-01**: ヒートマップの記事ありセルに Tooltip を表示できる（記事タイトル全文・サムネイル）— Phase 5で実装（D-05によりタイトル全文表示）
- [x] **TIP-02**: Tooltip クリックで記事ページへ遷移できる — Phase 5で実装
- [x] **TIP-03**: サムネイルは RSS `content:encoded` から取得し、未取得時はサムネイル領域省略 — Phase 5で実装（D-06によりプレースホルダーなし）

### UX — レイアウト・デザイン

- [x] **UX-01**: ビジュアル・レイアウトを 50 人規模対応にリファクタリングする — Phase 5で実装
