# Milestone v1.4: UI/UX Refresh

**Status:** ✅ SHIPPED 2026-05-15
**Phases:** 13-16
**Total Plans:** 4

## Overview

モバイルファーストを軸に、ファーストビュー・ヒートマップ・ポップオーバーのUIを刷新してユーザー体験を向上させる。

## Phases

### Phase 13: ファーストビュー + モバイルレイアウト

**Goal**: 開いた瞬間に継続データが目に飛び込むモバイルファーストのトップページを実現する
**Depends on**: Phase 12.1 (v1.3完了)
**Plans**: 1 plan

Plans:
- [x] 13-01: ヒーローバナー削除 + モバイル余白調整

**Details:**
ヒーローバナー画像を削除し、コンテナ余白をモバイル向け（px-3 py-4 pb-16）に縮小してファーストビューにヒートマップを表示。

---

### Phase 14: ユーザーリスト + チームタブ UI

**Goal**: ユーザーリスト行が1行で収まり、個人ページへの導線とタブ選択状態が直感的にわかる
**Depends on**: Phase 13
**Plans**: 1 plan

Plans:
- [x] 14-01: ユーザーリスト1行化 + シェブロン + タブ改善

**Details:**
HeatmapRowの名前列をtruncate1行化・シェブロン追加し、タブをSubstackオレンジ(#FF6719)に統一。

---

### Phase 15: ヒートマップ カラーリング

**Goal**: Substackオレンジの濃淡でメンバーの投稿状況が一目でわかるヒートマップになる
**Depends on**: Phase 14
**Plans**: 1 plan

Plans:
- [x] 15-01: ヒートマップ Substackオレンジ + 点線丸 + 投稿数表示

**Details:**
getIntensityClassをSubstackオレンジ系（#FF6719系）に変更し、投稿なしセルに点線丸・複数投稿セルに件数バッジを追加。Rippleエフェクトも追加。

---

### Phase 16: ポップオーバー刷新

**Goal**: 記事ポップオーバーが横並びレイアウト・ダーク背景で読みやすく、Click Outsideや×ボタンで直感的に閉じられるようになる
**Depends on**: Phase 15
**Plans**: 1 plan

Plans:
- [x] 16-01: ポップオーバー刷新（ダーク化 + 横並びレイアウト）

**Details:**
HeatmapTooltipをzinc-800ダーク背景・横並びサムネイルレイアウト・touchstart対応に刷新。Click Outside・×ボタン・Rippleエフェクト追加。

---

## Milestone Summary

**Key Decisions:**
- Rippleエフェクトを純CSSで実装（ライブラリ不要、KISS）
- ポップオーバーはClick+hover両対応（デスクトップ・モバイル共通）
- Substackブランドカラー #FF6719 を CSS変数 --color-primary として定数化
- line-clamp-2 はinline styleで実装（Tailwind v4でflex内line-clamp未対応）

**Issues Resolved:**
- ヒーローバナーによるファーストビュー占有
- 名前truncateなしでの行高さ不揃い問題
- ポップオーバーのモバイルタッチ操作非対応

**Technical Debt Incurred:**
- line-clamp-2 inline style → Tailwind v4アップデート時に再確認
- HeatmapTooltip: ライトテーマ化はv1.4マイルストーン後にQuick Taskで対応
