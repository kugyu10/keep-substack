# Plan 21-01: Redisクリーンアップ

**Phase:** 21 — Redisクリーンアップ
**Plan:** 21-01
**Wave:** 1
**Status:** draft

## Objective

Upstash Redis関連のコード・パッケージを完全に削除し、コードベースをSupabaseのみで動作させる。

## Requirements Addressed

- MIGRATE-04: @upstash/redis 削除・KVファイル廃止・コードベース整理

## Pre-implementation Notes

- `kvMembers.ts` と `kvArticles.ts` はすでにSupabaseを使用しており、Redisを一切使っていない
- `redis.ts` が唯一の `@upstash/redis` 使用箇所
- `redis.ts` 自体はどこからもimportされていない（kvMembers/kvArticlesがSupabase移行済みのため）
- kvMembers.ts → members.ts、kvArticles.ts → articles.ts にリネームしてimportを更新する

## Success Criteria

1. `@upstash/redis` がpackage.jsonから削除されており、`npm install` でインストールされない
2. `kvMembers.ts`・`kvArticles.ts`・`redis.ts` が削除されており、これらを参照するimportが存在しない
3. `npm run build` がエラーなく完了する

## Tasks

### Task 1: `src/lib/redis.ts` を削除する

```bash
rm src/lib/redis.ts
```

このファイルは `@upstash/redis` を import しているが、どこからも参照されていない。

### Task 3: `src/lib/kvArticles.ts` → `src/lib/articles.ts` にリネームする
