# Plan 13-01: ヒーローバナー削除 + モバイル余白調整

**Phase:** 13 — ファーストビュー + モバイルレイアウト
**Plan:** 13-01
**Status:** complete

## Objective

`src/app/page.tsx` からヒーローバナー画像を削除し、コンテナ余白・マージンをモバイル向けに縮小する。
これによりスマートフォン（375px幅）で開いた瞬間にヒートマップグリッドがファーストビューに表示されるようにする。

## Requirements Addressed

- LAYOUT-01: ヒーローバナーを削除し、開いた瞬間に継続データが目に入るようにする
- LAYOUT-02: モバイルファーストのレイアウトで、スマホ利用を主軸に置く

## Success Criteria

1. `src/app/page.tsx` を開いてもバナー画像が表示されない（`<img src={topLogo.src} ...>` タグが存在しない）
2. コンテナクラスが `px-3 py-4 pb-16` になっており、左右余白が12pxに縮小されている
3. `import topLogo from '@/data/top_logo.png'` の import 行が削除されている
4. `layout.tsx` は変更されていない（OGP参照を維持）

## Tasks

### Task 1: topLogo import を削除する

**File:** `src/app/page.tsx`

`import topLogo from '@/data/top_logo.png'` を削除する。

**Before:**
```typescript
import topLogo from '@/data/top_logo.png'
```

**After:** (行ごと削除)

---

### Task 2: ヒーローバナー `<img>` タグを削除する

**File:** `src/app/page.tsx`

バナー画像の `<img>` タグと eslint-disable コメントを削除する。

**Before:**
```tsx
{/* eslint-disable-next-line @next/next/no-img-element */}
<img src={topLogo.src} alt="Keep Substack" className="w-full h-auto object-contain mb-8 rounded" />
```

**After:** (2行ごと削除)

---

### Task 3: コンテナ余白をモバイル向けに縮小する

**File:** `src/app/page.tsx`

`<main>` のクラスを変更する。

**Before:**
```tsx
<main className="max-w-[600px] mx-auto p-6 pb-64">
```

**After:**
```tsx
<main className="max-w-[600px] mx-auto px-3 py-4 pb-16">
```

---

### Task 4: h1 下マージンを縮小する

**File:** `src/app/page.tsx`

h1 の `mb-4` を `mb-2` に変更してデータ領域を最大化する。

**Before:**
```tsx
<h1 className="text-2xl mb-4" style={{ fontFamily: 'Georgia, serif', fontWeight: 900 }}>Keep Substack</h1>
```

**After:**
```tsx
<h1 className="text-2xl mb-2" style={{ fontFamily: 'Georgia, serif', fontWeight: 900 }}>Keep Substack</h1>
```

---

### Task 5: チームタブ下マージンを縮小する

**File:** `src/app/page.tsx`

チームタブ wrapper div の `mb-6` を `mb-4` に変更する。

**Before:**
```tsx
<div className="flex gap-2 mb-6 flex-wrap">
```

**After:**
```tsx
<div className="flex gap-2 mb-4 flex-wrap">
```

---
