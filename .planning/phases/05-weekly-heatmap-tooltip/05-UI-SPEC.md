## Component Inventory

### 新規作成コンポーネント（D-11）

#### `WeeklyHeatmapGrid.tsx`

- Server Component または Client Component のラッパー
- Props: `results: { member: Member; items: FeedItem[] }[]`, `dates: string[]`（直近7日の `"YYYY-MM-DD"` 配列）
- 構造: 日付ヘッダー行 + HeatmapRow × メンバー数

#### `HeatmapRow.tsx`

- 1メンバー分のヒートマップ行
- Props: `member: Member`, `articlesByDate: Map<string, FeedItem[]>`, `dates: string[]`
- 左端: メンバー名 `w-32 truncate` リンク (`/member/{substackId}`) — ソース: D-10, HEAT-02
- 右側: 7列のセルグリッド（各セルに色クラス適用）

#### `HeatmapCell.tsx`（任意、HeatmapRow 内に inline 定義も可）

- 1セル
- 記事あり → hover/tap で Tooltip 表示
- 記事なし → `bg-gray-100` のみ（Tooltip なし）

#### `HeatmapTooltip.tsx`

- ArticleTooltip.tsx のパターンを踏襲した新 Tooltip
- Props: `articles: { title: string; link: string; thumbnail?: string }[]`
- デスクトップ: hover でオープン / モバイル: tap でトグル — ソース: D-07
- `useRef` + `useEffect` で click outside 検知（ArticleTooltip.tsx パターン継承）
- `'use client'` 必須

### 既存コンポーネント（変更禁止）

- `MiniCalendar.tsx` — `/member/` ページで継続使用。本フェーズでは改造しない — ソース: D-12
- `ArticleTooltip.tsx` — 参照実装として読むが改造しない

## Layout Contract

### ページ全体（page.tsx）

```
<main class="max-w-6xl mx-auto p-6">
  <h1>Keep Substack</h1>          ← text-2xl font-semibold mb-6
  <WeeklyHeatmapGrid ... />
</main>
```

### WeeklyHeatmapGrid の構造

```
<div>
  <!-- 日付ヘッダー行 -->
  <div class="flex">
    <div class="w-32 shrink-0" />    ← メンバー名列の幅確保（空白）
    <div class="grid grid-cols-7 flex-1">
      {dates.map(d => <span class="text-xs text-center">MM/DD</span>)}
    </div>
  </div>
  <!-- メンバー行 (1行 = 1メンバー) -->
  {members.map(m => <HeatmapRow ... />)}

```

### HeatmapRow の構造

```
<div class="flex items-center border-b border-gray-100 py-1">
  <a href="/member/{substackId}" class="w-32 shrink-0 truncate text-sm font-semibold pr-2">
    {member.name}
  </a>
  <div class="grid grid-cols-7 gap-1 flex-1">
    {dates.map(date => <HeatmapCell ... />)}
  </div>
</div>
```

### HeatmapTooltip の配置

```
<div class="relative">           ← セルの位置基準
  {open && (
    <div class="absolute bottom-full left-1/2 -translate-x-1/2 pb-2 z-10">
      <div class="bg-white border border-gray-200 rounded shadow-lg p-2 max-w-xs">
        {thumbnail && <img src={thumbnail} class="w-full rounded mb-1 object-cover max-h-24" />}
        <a href={withUtm(link)} class="text-xs text-blue-600 hover:underline break-words block">
          {title}
        </a>
      </div>
    </div>
  )}
  <button ...>{/* セルの中身 */}</button>
</div>
```

スクロール方針: 縦スクロールのみ、スティッキーヘッダーなし — ソース: D-09

## Interaction Contract

### ヒートマップセル — 記事あり

| 状態 | 表示 |
|------|------|
| 通常 | 投稿数に応じた green 背景セル |
| Desktop: hover | Tooltip オープン（onMouseEnter） |
| Desktop: hover-out | Tooltip クローズ（onMouseLeave、Tooltip 内へ移動中は維持） |
| Mobile: tap | Tooltip トグル（open/close） |
| Mobile: outside tap | Tooltip クローズ（click outside 検知） |

### ヒートマップセル — 記事なし

| 状態 | 表示 |
|------|------|
| 通常 | `bg-gray-100` のみ |
| hover/tap | 無反応（Tooltip なし、cursor-default） |

### メンバー名リンク

- クリックで `/member/{substackId}` へ遷移（Next.js `<Link>`）
- hover で `underline`

### Tooltip 内記事リンク

- クリックで記事ページへ `target="_blank" rel="noopener noreferrer"` — ソース: TIP-02
- UTM パラメータ付与: `utm_source=keep-substack&utm_medium=referral`（ArticleTooltip.tsx パターン継承）

## Thumbnail Contract

| 条件 | 表示 |
|------|------|
| `content:encoded` に `<img src="...">` あり | 最初の `img src` を thumbnail として表示 |
| thumbnail なし | サムネイル領域ごと省略、タイトルのみ表示（プレースホルダーなし）— ソース: D-06 |

サムネイル表示スタイル:

```
<img
  src={thumbnail}
  alt=""
  class="w-full rounded mb-1 object-cover max-h-24"
/>
```

- `max-h-24`（96px）でタテ方向を制限
- `object-cover` でアスペクト比を維持してクロップ
- `alt=""` — 装飾的画像のため空（アクセシビリティ考慮）

## Date Header Contract

- 形式: `MM/DD`（例: `5/3`, `5/4`）— ソース: D-13
- 算出: `new Date()` から today を起点に 6日前まで降順で7要素の `"YYYY-MM-DD"` 配列を生成
- 表示: 配列順（左: 6日前 → 右: 今日）または（左: 今日 → 右: 6日前）はClaude裁量
- ヘッダーはそれぞれのセル列の真上に配置（`grid-cols-7` グリッドで自動整列）

---

## Sort Contract

ソート基準 — ソース: D-09, HEAT-03:

1. 直近7日間の投稿数: 降順
2. 同数の場合: `member.addedAt` 昇順

## Accessibility Notes

- Tooltip をトリガーするセルには `aria-expanded={open}` を付与（ArticleTooltip.tsx パターン継承）
- Tooltip 内クローズボタン: `aria-label="閉じる"`
- サムネイル画像: `alt=""` (装飾的画像)
- メンバー名リンク: テキスト自体がラベル
- セルはボタン要素（`<button>`）でキーボードアクセス可能
