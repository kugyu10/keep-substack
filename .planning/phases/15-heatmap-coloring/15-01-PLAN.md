# Plan 15-01: ヒートマップ Substackオレンジ + 点線丸 + 投稿数表示

**Phase:** 15 — ヒートマップ カラーリング
**Plan:** 15-01
**Status:** complete

## Objective

`src/lib/heatmapUtils.ts` の `getIntensityClass` をSubstackオレンジ系に変更し、
`src/components/HeatmapRow.tsx` の投稿なし日付セルに点線丸を追加、複数投稿セルに投稿数を表示する。

## Requirements Addressed

- HEATMAP-01: Substackオレンジ（#FF6719系）の濃淡で投稿数を表現する
- HEATMAP-02: 投稿なしは点線丸、投稿ありはオレンジベタ塗り（複数投稿は数字表示）

## Success Criteria

1. 投稿ありセルが `bg-orange-*` / `bg-[#FF6719]` 系のオレンジで表示される
2. 投稿数が多いほど濃いオレンジになる（1件=薄→5件以上=最濃）
3. 投稿なしセルが点線丸（`border-dashed border-gray-300 rounded-full`）で表示される
4. 投稿数が2件以上のセルに件数の数字が表示される
5. `npm run build` がエラーなく完了すること

## Tasks

### Task 1: `getIntensityClass` をSubstackオレンジ系に変更する（HEATMAP-01）

**File:** `src/lib/heatmapUtils.ts`

`getIntensityClass` のカラークラスをグリーン系からオレンジ系に変更する。

**Before:**
```ts
export function getIntensityClass(count: number): string {
  if (count === 0) return 'bg-gray-100 text-gray-400'
  if (count === 1) return 'bg-green-100 text-green-700'
  if (count === 2) return 'bg-green-200 text-green-800'
  if (count === 3) return 'bg-green-300 text-green-900'
  if (count === 4) return 'bg-green-500 text-white'
  return 'bg-green-700 text-white'
}
```

**After:**
```ts
export function getIntensityClass(count: number): string {
  if (count === 1) return 'bg-orange-200 text-orange-950'
  if (count === 2) return 'bg-orange-300 text-orange-950'
  if (count === 3) return 'bg-[#FF6719] text-white'
  if (count === 4) return 'bg-orange-600 text-white'
  return 'bg-orange-800 text-white'
}
```

注意: `count === 0` のケースを削除する（呼び出し元 HeatmapRow.tsx で count > 0 の場合のみ呼ばれるため不要）。

---

### Task 2: 投稿なし日付セルに点線丸を追加する（HEATMAP-02）

**File:** `src/components/HeatmapRow.tsx`

投稿なし日付セルに点線丸スタイルを追加する。

**Before:**
```tsx
          if (count === 0) {
            return <div key={date} className="aspect-square" />
          }
```

**After:**
```tsx
          if (count === 0) {
            return <div key={date} className="aspect-square rounded-full border border-dashed border-gray-300" />
          }
```

---

### Task 3: 複数投稿セルに投稿数を表示する（HEATMAP-02）

**File:** `src/components/HeatmapRow.tsx`

`HeatmapTooltip` に `children` として投稿数を渡す。`count >= 2` の場合のみ表示。

**Before:**
```tsx
          return (
            <HeatmapTooltip
              key={date}
              articles={articles}
              colorClass={getIntensityClass(count)}
              imageUrl={imageUrl}
            />
          )
```

**After:**
```tsx
          return (
            <HeatmapTooltip
              key={date}
              articles={articles}
              colorClass={getIntensityClass(count)}
              imageUrl={imageUrl}
            >
              {count >= 2 ? <span className="text-xs font-bold leading-none">{count}</span> : null}
            </HeatmapTooltip>
          )
```

## Notes

- `HeatmapTooltip` の button は既に `{children}` を `flex items-center justify-center` で中央表示しているため、count span を渡すだけでよい
- `count === 0` は HeatmapRow.tsx で先にガードされるため `getIntensityClass` に渡されることはない
- `bg-[#FF6719]` は Tailwind v4 arbitrary value 記法
- `HeatmapTooltip.tsx` は変更しない
