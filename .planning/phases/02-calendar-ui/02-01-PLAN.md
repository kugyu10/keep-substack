新規ファイル `src/lib/calendarUtils.ts` を作成する。3つの関数と2つの型を export する。

**DayInfo 型:**
```typescript
export type DayInfo = {
  date: number       // 1-31
  colStart?: number  // 月初日のみ: 1（日曜）〜7（土曜）
}
```

**ArticleMap 型:**
```typescript
export type ArticleMap = Map<string, { title?: string; link?: string }[]>
// key は "YYYY-MM-DD" 形式
```

**parseIsoDate 関数:**
- `isoDate.match(/^(\d{4})-(\d{2})-(\d{2})/)` で文字列から直接パースする
- `new Date(isoDate).getDate()` は使わない（タイムゾーン依存バグを防ぐため）
- マッチしない場合は null を返す

**buildDayGrid 関数:**
- 引数: `(year: number, month: number): DayInfo[]`
- `new Date(year, month - 1, 1).getDay()` で月初の曜日を取得（0=日曜）
- colStart = getDay() + 1（1始まりにする）
- 月の最終日は `new Date(year, month, 0).getDate()` で取得
- 1日目に `colStart` をセット、2日目以降は `colStart` なし

**buildArticleMap 関数:**
- 引数: `(items: { title?: string; link?: string; isoDate?: string }[]): ArticleMap`
- isoDate がない item はスキップ
- parseIsoDate で変換できない item はスキップ
- key を `"YYYY-MM-DD"` 形式でゼロパディングして生成
- 同一日に複数記事がある場合は配列に push して全件保持（D-02 per 02-CONTEXT.md）

  <name>Task 2: CalendarGrid.tsx — 月ナビゲーション付きカレンダーグリッド（Client Component）</name>
  <files>src/components/CalendarGrid.tsx</files>
  <action>
新規ファイル `src/components/CalendarGrid.tsx` を作成する。ファイル先頭に `'use client'` を付与すること。

**Props 型（同ファイルに定義）:**
```typescript
type Props = {
  memberName: string
  articleMap: [string, { title?: string; link?: string }[]][]
  // ArticleMap は Map のためシリアライズ不可。Server→Client 渡しは配列で行う
}
```

**コンポーネント構成:**
1. `useState` で `{ year, month }` を管理。初期値は `new Date()` の現在年月
2. 月ナビゲーション UI:
   - `<button onClick={prevMonth}>＜</button>` — 前月へ
   - `<span>{year}年{month}月</span>` — 現在月表示
   - `<button onClick={nextMonth}>＞</button>` — 次月へ
   - prevMonth: month === 1 なら year-1/month=12、それ以外は month-1
   - nextMonth: month === 12 なら year+1/month=1、それ以外は month+1
3. 曜日ヘッダー行: `['日','月','火','水','木','金','土']` を横並び7列で表示
4. 日付グリッド:
   - `buildDayGrid(year, month)` を呼び出して DayInfo[] を取得
   - `new Map(articleMapEntries)` で props の配列から Map を再構築
   - key を `YYYY-MM-DD` 形式で生成（月・日はゼロパディング）して articleMap を検索
   - 月初日の DayCell は `style={{ gridColumnStart: day.colStart }}` を付与する（Tailwind の動的クラス `col-start-${n}` は JIT で検知されないため style prop を使う）

**セルのスタイル（CAL-02: Claude裁量）:**
- 記事あり: `bg-green-500 text-white font-semibold rounded hover:bg-green-600 transition-colors` — GitHubの草風の緑で「頑張り」が一目でわかる
- 記事なし: `text-gray-500 rounded hover:bg-gray-100`
- 共通: `aspect-square flex items-center justify-center text-sm w-full h-full`

**全体レイアウト:**
```tsx
<div>
  <h2 className="text-lg font-semibold mb-2">{memberName}</h2>
  {/* ナビゲーション */}
  <div className="flex items-center justify-between mb-3">...</div>
  {/* グリッド: grid grid-cols-7 gap-1 */}
  <div className="grid grid-cols-7 gap-1">
    {/* 曜日ヘッダー */}
    {/* 日付セル */}
  </div>
</div>
```
