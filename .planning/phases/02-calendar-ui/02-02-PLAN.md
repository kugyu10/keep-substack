新規ファイル `src/components/ArticleTooltip.tsx` を作成する。ファイル先頭に `'use client'` を付与すること。

**目的:** セルに重ねる形で、記事リストをセルの上方向に表示するツールチップ。ホバーとクリックの両方で開き、外クリックまたは×ボタンで閉じる（D-01, D-02, D-03 per 02-CONTEXT.md）。

**Props 型:**
```typescript
type Article = { title?: string; link?: string }

type Props = {
  date: number
  articles: Article[]
  colStart?: number  // 月初日のみ使用
}
```

**状態管理:**
- `useState<boolean>(false)` でツールチップ開閉を管理（変数名: `open`）
- `useRef<HTMLDivElement>(null)` でセルの DOM ノードを参照（変数名: `cellRef`）

**イベントハンドリング:**
- `onMouseEnter`: `setOpen(true)`
- `onMouseLeave(e)`: `e.relatedTarget` が `cellRef.current` 内に含まれるかチェックし、含まれる場合は閉じない。含まれない場合は `setOpen(false)`
- `onClick`: `setOpen(prev => !prev)` でトグル（ホバーできないモバイル対応）

**外クリック検知（useEffect）:**
```typescript
useEffect(() => {
  if (!open) return
  function handleClickOutside(e: MouseEvent) {
    if (cellRef.current && !cellRef.current.contains(e.target as Node)) {
      setOpen(false)
    }
  }
  document.addEventListener('mousedown', handleClickOutside)
  return () => document.removeEventListener('mousedown', handleClickOutside)
}, [open])
```

**ツールチップ本体の配置（D-03: セル上方向）:**
```tsx
{open && (
  <div
    className="absolute bottom-full left-1/2 -translate-x-1/2 mb-2 z-10 bg-white border border-gray-200 rounded shadow-lg p-2 min-w-max max-w-xs"
    onMouseEnter={() => setOpen(true)}
    onMouseLeave={() => setOpen(false)}
  >
    {/* ×ボタン */}
    <button
      className="absolute top-1 right-1 text-gray-400 hover:text-gray-600 text-xs"
      onClick={() => setOpen(false)}
      aria-label="閉じる"
    >
      ×
    </button>
    {/* 記事リスト（D-02: 全件表示） */}
    <ul className="space-y-1 pr-4">
      {articles.map((a, i) => (
        <li key={i}>
          <a
            href={a.link ?? '#'}
            target="_blank"
            rel="noopener noreferrer"
            className="text-xs text-blue-600 hover:underline block"
          >
            {a.title ?? '(タイトルなし)'}
          </a>
        </li>
      ))}
    </ul>
  </div>
)}
```

**セル本体のスタイル（記事あり）:**
```tsx
<button
  className="w-full h-full flex items-center justify-center text-sm font-semibold bg-green-500 text-white rounded hover:bg-green-600 transition-colors"
  onMouseEnter={() => setOpen(true)}
  onMouseLeave={...}
  onClick={() => setOpen(prev => !prev)}
  aria-expanded={open}
>
  {date}
</button>
```

**ルートコンテナ:**
```tsx
<div
  ref={cellRef}
  style={colStart ? { gridColumnStart: colStart } : undefined}
  className="relative aspect-square"
>
  ...
</div>
```

**注意事項:**
- `target="_blank"` のリンクは必ず `rel="noopener noreferrer"` を付与すること（タブナプジャック防止）
- ツールチップは `position: absolute; bottom: 100%` でセルの上に表示（D-03）
- ツールチップ自体に `onMouseEnter/Leave` を付与して、ツールチップ内にカーソルが入っても閉じないようにする

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| RSSフィード → ツールチップ表示 | 外部フィードの title と link が XSS や不正 URL の可能性 |
| Client Component → 外部サイト | `target="_blank"` リンクからのタブナプジャック |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-04 | Tampering | ArticleTooltip.tsx 外部リンク | mitigate | `target="_blank"` の全リンクに `rel="noopener noreferrer"` を付与する（タブナプジャック防止） |
| T-02-05 | Tampering | ArticleTooltip.tsx リンク先 URL | accept | `href={a.link ?? '#'}` のみ使用。React は href をエスケープするため XSS リスクは低い。`javascript:` スキームは今回の RSS データ（Substack）では発生しない想定 |
| T-02-06 | Denial of Service | ArticleTooltip.tsx useEffect | mitigate | useEffect の cleanup で `removeEventListener` を必ず呼ぶことでリスナー積み重なりを防止 |
</threat_model>
