### Task 2: 名前を1行truncateに変更する（LIST-01）

**File:** `src/components/HeatmapRow.tsx`

名前 div の `hidden sm:block` を削除して全サイズで表示、`-webkit-line-clamp` を廃止して `truncate` に変更する。

**Before:**
```tsx
{/* Tailwind v4でline-clamp-2がflex内で効かないためinline styleで対応 */}
<div
  className="hidden sm:block text-xs font-semibold leading-snug min-w-0 underline"
  style={{ overflow: 'hidden', display: '-webkit-box', WebkitBoxOrient: 'vertical', WebkitLineClamp: 2 }}
>
  {member.name}
</div>
```

**After:**
```tsx
<div className="flex-1 min-w-0 text-xs font-semibold leading-snug underline truncate">
  {member.name}
</div>
```

注意: `truncate` = `overflow-hidden whitespace-nowrap text-overflow-ellipsis`。Tailwind v4でflex内でも動作する（`flex-1 min-w-0` が必要）。`hidden sm:block` は削除して全サイズで表示する。

### Task 3: シェブロンを追加する（LIST-02）

**File:** `src/components/HeatmapRow.tsx`

名前 div の直後、`</Link>` の直前にシェブロン `›` を追加する。

**Before:**
```tsx
        <div className="flex-1 min-w-0 text-xs font-semibold leading-snug underline truncate">
          {member.name}
        </div>
      </Link>
```

**After:**
```tsx
        <div className="flex-1 min-w-0 text-xs font-semibold leading-snug underline truncate hidden sm:block">
          {member.name}
        </div>
        <span className="shrink-0 text-gray-400 text-xs" aria-hidden="true">›</span>
      </Link>
```

### Task 4: アクティブタブをSubstackオレンジに変更する（LIST-03）

**File:** `src/app/page.tsx`

「All」タブとチームタブ両方のアクティブスタイルを変更する。

**All タブ Before:**

```tsx
}`}
```

**チームタブ After:**
```tsx
className={`px-3 py-1 rounded text-sm border ${
  team === t ? 'bg-[#FF6719] text-white border-[#FF6719]' : 'text-gray-600 border-gray-300 hover:bg-gray-50'
```
