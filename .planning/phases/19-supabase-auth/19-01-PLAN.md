# Plan 19-01: /login ページ + Magic Link + /auth/callback

**Phase:** 19 — Supabase Auth + メンバー自己管理
**Plan:** 19-01
**Wave:** 1
**Status:** draft

## Objective

Magic Link 認証のエントリポイントを実装する。
`/login` ページでメールアドレスを入力すると Magic Link メールが送信され、
メール内のリンクをクリックすると `/auth/callback` を経由してログイン状態になり `/my` にリダイレクトされる。

## Requirements Addressed

- AUTH-01: /loginページでメールアドレスを入力してMagic Linkを送信できる
- AUTH-02 前提: メール内のリンクをクリックするとログイン状態になり/myページにリダイレクトされる（SC-2）

## Pre-implementation Notes

- `@supabase/ssr` は既にインストール済み（package.json `^0.10.3`）
- `createSupabaseServerClient()` は `src/lib/supabase/server.ts` に存在する（cookies 管理あり）
- Magic Link の送信には `supabase.auth.signInWithOtp({ email, options: { emailRedirectTo } })` を使う
- Supabase Dashboard で **Email Redirect URLs** に `/auth/callback` を登録する必要がある（人間作業）
- `/auth/callback` は PKCE フロー: `exchangeCodeForSession(code)` → セッションCookieをセット → `/my` にリダイレクト
- ログイン済みユーザーが `/login` にアクセスした場合は `/my` にリダイレクトする

## Prerequisite: Supabase Dashboard の設定

**このタスクはコード変更と並行してユーザーが実施する必要がある。**

Supabase Dashboard > Authentication > URL Configuration:
- **Site URL**: `https://your-app.vercel.app`（または localhost:3000）
- **Redirect URLs** に追加: `https://your-app.vercel.app/auth/callback`

## Success Criteria

1. `/login` ページが表示され、メールアドレス入力フォームがある
2. メールを送信すると「メールを送信しました」のメッセージが表示される
3. Magic Link メールが届き、クリックすると `/auth/callback` を経由して `/my` にリダイレクトされる
4. `npm run build` がエラーなく完了する

## Tasks

### Task 1: `src/app/login/actions.ts` を作成する

**File:** `src/app/login/actions.ts`（新規作成）

```typescript
'use server'

import { createSupabaseServerClient } from '@/lib/supabase/server'
import { headers } from 'next/headers'

export async function sendMagicLinkAction(
  prevState: string | null,
  formData: FormData
): Promise<string | null> {
  const email = formData.get('email') as string
  if (!email) return 'メールアドレスを入力してください'

  const headersList = await headers()
  const origin = headersList.get('origin') ?? ''

  const supabase = await createSupabaseServerClient()
  const { error } = await supabase.auth.signInWithOtp({
    email,
    options: {
      emailRedirectTo: `${origin}/auth/callback`,
    },
  })

  if (error) {
    console.error('[sendMagicLink]', error)
    return 'メールの送信に失敗しました。しばらく経ってから再試行してください'
  }

  return 'SENT'
}
```

### Task 2: `src/app/login/LoginForm.tsx` を作成する

**File:** `src/app/login/LoginForm.tsx`（新規作成）

Client Component。useActionState でメール送信と送信済み表示を管理する。

```typescript
'use client'

import { useActionState } from 'react'
import { sendMagicLinkAction } from './actions'

export default function LoginForm() {
  const [state, action, isPending] = useActionState(sendMagicLinkAction, null)

  if (state === 'SENT') {
    return (
      <div className="text-center py-8">
        <p className="text-lg font-medium mb-2">メールを送信しました</p>
        <p className="text-sm text-gray-600">
          受信トレイを確認してログインリンクをクリックしてください
        </p>
      </div>
    )
  }

  return (
    <form action={action} className="space-y-4">
      {state && state !== 'SENT' && (
        <p className="text-sm text-red-600">{state}</p>
      )}
      <div>
        <label htmlFor="email" className="block text-sm font-medium mb-1">
          メールアドレス
        </label>
        <input
          id="email"
          name="email"
          type="email"
          required
          autoComplete="email"
          className="w-full border rounded px-3 py-2 text-sm"
          placeholder="you@example.com"
        />
      </div>
      <button
        type="submit"
        disabled={isPending}
        className="w-full bg-orange-500 text-white rounded px-4 py-2 text-sm font-medium disabled:opacity-50"
      >
        {isPending ? '送信中...' : 'ログインリンクを送信'}
      </button>
    </form>
  )
}
```

### Task 4: `src/app/auth/callback/route.ts` を作成する

**File:** `src/app/auth/callback/route.ts`（新規作成）

PKCE フローの code exchange を処理する Route Handler。

```typescript
import { NextRequest, NextResponse } from 'next/server'
import { createSupabaseServerClient } from '@/lib/supabase/server'

export async function GET(request: NextRequest) {
  const { searchParams, origin } = new URL(request.url)
  const code = searchParams.get('code')
  const next = searchParams.get('next') ?? '/my'

  if (code) {
    const supabase = await createSupabaseServerClient()
    const { error } = await supabase.auth.exchangeCodeForSession(code)
    if (!error) {
      return NextResponse.redirect(new URL(next, origin))
    }
    console.error('[auth/callback] exchangeCodeForSession error:', error)
  }

  // エラー時: /login にリダイレクト
  return NextResponse.redirect(new URL('/login', origin))
}
```
