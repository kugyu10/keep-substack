## ハイブリッドフェッチの動作

```
fetchAllFeedsCached(members)
  └─ Promise.allSettled (外側: メンバー並列)
       └─ 各メンバーごとに Promise.allSettled (内側: RSS/KV並列)
            ├─ fetchWithRetry(feedUrl)  ← ライブRSS
            └─ getArticles(substackId) ← KV過去記事

  フォールバックロジック:
  - live 成功 + kv 成功 → マージ（linkでdedupe → isoDate降順ソート）
  - live 失敗 + kv 成功 → KVのみ
  - live 成功 + kv 失敗 → ライブRSSのみ
  - 両方失敗 → { items: [], imageUrl: undefined }

  imageUrl: live.imageUrl ?? kv.imageUrl（ライブRSS優先）
  dedupe: link が undefined の記事は Set.has(undefined) 問題を回避してすべて含む
```
