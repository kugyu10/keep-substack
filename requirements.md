# 指示書
Next.js (App Router) と Tailwind CSS を使って、特定のRSSフィードの更新頻度をカレンダー形式で確認できるWebアプリを作ってください。

## 要件:

- rss-parser を使用して、サーバーサイドで複数のRSSフィード（例: [フィードURL1, フィードURL2]）を取得・解析してください。

- 取得した記事の「公開日（pubDate）」データを元に、カレンダーUI（またはGitHubのコントリビューショングラフのようなUI）を作成してください。

- カレンダーの各日付セルには、その日に更新があった場合は「〇」や色を付け、誰が更新したかが分かるようにホバーで記事タイトルやブログ名を表示してください。

- モダンで見やすいデザインにしてください。まずは1つのページ（page.tsx）にすべてまとめて記述してください。

- 不明な点は必ずヒアリングしてください
